## Updating

### Android

Please find the download link in the [above section](#android)

Please note that, Google Play Store version and sideloaded version is not compatible with each other due to its signing key. If you are installing the app from Google Play, then update it via Google Play too.

### Windows

For winget, run the following command to update to the latest release.

```powershell
winget upgrade MuhammadFareezIqmal.IIUMSchedule
```

For other types of installation, see above [section](#windows).

### Web

Visit the link in the [above section](#web).

Browser tends to cache the current version for a some duration before load a new version.
To force update, you can use hot refresh (<kbd>Ctrl+F5</kbd>) in desktop browser.
